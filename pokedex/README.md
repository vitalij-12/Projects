# Kanto Single-Typ Basis-Pokédex

Ein kleines Projekt, das die **Basis-Pokémon der Kanto-Region mit nur einem Typ** als Pokédex enthält.  

**Enthaltene Pokémon:** Bisasam, Glumanda, Schiggy, Raupy, Hornliu, Rattfratz, Pikachu, Sandan, Nidoran♀, Nidoran♂, Vulpix, Digda, Mauzi, Enton, Menki, Fukano, Quapsel, Sterndu, Relaxo, Karpador  

Daten umfassen Name, Typ, Fähigkeiten, Größe, Gewicht und Geschlecht.
